# Movies-to-Watch

Keep track of what movies you wish to watch, and have watched.

* Search for a movie to pull up movie info.
* Add a movie from the search section or type one in to watch.
* Click "watched" to set the movie to the watched section.
* Edit a specific entry in case it was misspelled.
* Delete a movie from your list.

This was build using HTML, CSS, Handlebars, Javascript, Node.js(server), MYSQL database, and node packages(body-parser, express, mysql, and request.)
